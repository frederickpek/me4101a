Graph Drawing Tool


  1. Features
    1. Left-click to create vertices.
    2. Click on vertices to create edges, click on another vertex to end the edge creation.
    3. Rick-click to delete vertices and edges.
    4. Use the scoller buttom to click, drag and drop vertices.
    5. Press Ctrl + Z to undo.
    6. Press Ctrl + Y to redo.

  2. Pending
